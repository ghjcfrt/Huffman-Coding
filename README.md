# 运用哈弗曼编码进行压缩、解压

题目描述： 设计一个哈夫曼编码、解码系统，用于对文本或图像进行压缩和解压缩。项目包括以下功能：

文件读取：从文件中读取文本或图像数据。

频率分析：计算每个字符或像素的频率。

哈夫曼编码：根据频率构建哈夫曼树，并生成编码表。

压缩数据：使用哈夫曼编码对数据进行压缩。

解码数据：根据哈夫曼编码表解压缩数据。

文件保存：将压缩或解压缩后的数据保存到文件。

知识点：

数据结构：树（如哈夫曼树）

算法：哈夫曼编码、解码


## 使用方法
下载___Huffman_Encoding.Decoding.exe___后，运行 ___Huffman_Encoding.Decoding.exe___
